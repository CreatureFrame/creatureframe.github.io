Set of animations for an alpaca character, for use with a 3rd person player controller in Unity. Entirely hand animated.


Running in Unity environment.



Instead of using cloth simulation, the poncho was manually smoothed with blend shapes to create a less busy look, referencing how real woollen ponchos behave in that they stretch and wrap to the wearer’s shoulders.


Complete animation lineup exported to Unity:



Interactive Sketchfab Controls:
Move camera: 1 finger drag or Left Mouse Button
Pan: 2-finger drag or Right Mouse Button or SHIFT+ Left Mouse Button
Zoom on object: Double-tap or Double-click on object
Zoom out: Double-tap or Double-click on background
Zoom: Pinch in/out or Mousewheel or CTRL + Left Mouse Button
Mesh wireframe can be turned on from the settings.











Rig was set up to allow for picking up and carrying around props in Unity controller.
